CDSS - Clinical Decision Support System
Overview
This project was developed by Rameesha Amir. It is a modern Clinical Decision Support System (CDSS) Frontend UI designed to help healthcare professionals manage patient data and view clinical insights efficiently.  

Tech Stack
Structure: HTML5   

Styling: Tailwind CSS (Loaded via CDN)   

Logic: Vanilla JavaScript for navigation and login simulation   

Key Features
The project includes the following core modules:

Login Page: A secure entry point for healthcare providers.  

Dashboard: A central hub for viewing patient schedules and clinical alerts.  

Patient Form: An interface for entering vital signs and diagnostic data.  

Results View: Displays clinical analysis and predictive risk assessments.  

Reports: System analytics and historical clinical data.  

Settings: Account and system configuration options.  

Deployment and Usage
To run the project, follow these steps:

Download: Ensure all project files are saved in a single folder.  

Open: Double-click the index.html file to launch the application in your web browser.  

Requirements: An active internet connection is required to load the Tailwind CSS styles.  

Technical Notes
Frontend UI Only: This project is a user interface prototype; it does not include a backend or a database.
